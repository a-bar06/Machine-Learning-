##  Overview
This project analyzes student academic performance using Machine Learning techniques.

The goal is to identify patterns in student data and predict performance outcomes based on features such as:
- Study time
- Attendance
- Scores
- Other academic factors

---

## Objectives
- Clean and preprocess student dataset
- Perform Exploratory Data Analysis (EDA)
- Visualize performance trends
- Train Machine Learning models
- Evaluate model performance using appropriate metrics
